sql的函数在sql中起到举足轻重的地位，在sql语句中使用好函数，能减轻语句的臃肿，甚至还能起到优化sql语句的作用
本篇主要记录一些sql常用函数的知识点

一、字符函数

1、 LENGTH

获取参数值的字节个数

2、 CONCAT

拼接字符串

3、 UPPER

将参数全部变为大写

4、 LOWER

将参数全部变为小写

5、 SUBSTR

索引从1开始，返回指定位置及其之后的字符串

截取从指定索引处（第二个参数）开始，获取索引位置及其后指定字符长度的字符串（第三个参数）

6、 INSTR

返回第二个参数在第一个参数中第一次出现的位置，若没有出现，则返回0

7、 TRIM

去除首尾指定的字符，若没有指定，则表示去除空格

扩展：也可使用LTRIM和RTRIM函数
SELECT LTRIM(‘ ## ‘); | SELECT RTRIM(‘ ## ‘);
结果：’## ‘ | ‘ ##’

8、 LPAD

用指定的字符实现左填充指定长度，同时也是结果的最终长度

9、 RPAD

用指定的字符实现右填充指定长度，同时也是结果的最终长度

10、 REPLACE

替换所有的指定的字符串为另一个字符串

二、数学函数

1、 ROUND

四舍五入

保留指定小数的位数

2、 CEIL

向上取整，返回大于等于该参数的最小整数

3、 FLOOR

向下取整，返回小于等于该参数的最大整数

4、 TRUNCATE

截断

5、 MOD

取模

三、日期函数

1、 NOW

返回当前系统日期+时间

2、 CURDATE

返回当前系统时间，不包含时间

3、 CURTIME

返回当前时间，不包含日期

4、 MONTHNAME , DAYNAME 等

返回日期的英文名字

5、 STR_TO_DATE

将字符串通过指定的格式转化为日期

6、 DATE_FORMAT

将日期转化为指定格式字符串

三、其他函数

##1、 SELECT VERSION()
获取sql的版本号

2、 SELECT DATABASE()

获取当前所在数据库名

3、 SELECT USER()

获取登陆用户名

四、流程控制函数

1、 if

一共三个参数，若第一个参数代表的表达式值为true, 则执行第二个参数代表的表达式，否则执行第三个参数代表的表达式

2、 case

用法一：
相当于switch case

用法二：
相当于多重if
相当于switch case
